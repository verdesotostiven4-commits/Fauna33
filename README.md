# Fauna33 — Clase Amphibia

Esquema web interactivo de la clase Amphibia, elaborado únicamente con la información del PDF proporcionado.

## Reglas del contenido

- Órdenes ordenados alfabéticamente.
- Familias ordenadas alfabéticamente dentro de cada orden.
- Especies representativas y endémicas extraídas del PDF.
- Nombres científicos en cursiva, sin negrita ni subrayado.

## Ejecutar localmente

```bash
npm install
npm run dev
```

## Deploy en Vercel

- Framework: Vite
- Build command: `npm run build`
- Output directory: `dist`
